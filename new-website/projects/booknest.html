BookNest Web Application in Java & Spring Boot

  • Built a full-stack web app for tracking books, enabling users to search books via the Open Library API and interact with DataStax Cassandra for database management.
  • Implemented OAuth2-based GitHub login for secure user authentication, enabling personalized features such as adding books to a "My Books" page.
  • Enabled users to track reading progress by recording the start and finish dates for each book.
  • Designed and implemented the backend using Spring Boot, ensuring scalability and efficient data handling.
  • Utilized a modular design with Java and XML for API interactions, while employing HTML for the front-end interface.
  • Leveraged Cassandra's NoSQL database to store and retrieve complex data structures such as book lists, author information, and user book-tracking data.
